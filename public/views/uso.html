Cardify

Cardify Es un plugin de jQuery que dado un contenedor busca las imagenes, envuelve en figure y agrega figcaption dentro de este el alt de la imagen al pasar el mouse muestra el figcaption con el alt centrado y oculta la imagen

Uso

Primer paso: descargar cardify

Descarga cardify

Segundo paso: incluir script en el html

jquery.cardify.css
jquery.cardify.js

Incluir script que vincule JS:

<script src="js/jquery.cardify.js"></script>
